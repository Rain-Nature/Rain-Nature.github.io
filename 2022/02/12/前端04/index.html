CSS基础

1. CSS简介

​ CSS(层叠样式表)是由W3C的CSS工作组创建和维护的。它是一种不需要编译。可直接由浏览器执行的标记性语言，用于控制web页面的外观

​ 通过使用CSS样式控制页面各元素的属性显示，可将页面的内容与表现形式进行分离

​ 层叠样式表本质上也是文本，用{} 将样式内容括住，最后将所有的封装在一个文件里。页面引用这个文件即可

​ HTML5和CSS3是同时制作发布的

2. CSS的优势

  • 避免使用多余的HTML代码，减少代码量

    ​ 文本本身是通过字符来计算文件量，而**在之前对于页面的修饰会通过HTML标签自带的属性进行修饰(即HTML代码的处理方式，不属于CSS)**，此时代码量大，读的时候就会慢

  • 缩短更新和维护的时间

    ​ 不使用CSS可能有些样式要反复定义，需要更换时，可能非常复杂。而且采用样式表可以使得样式与页面相分离，在修改时，更加直观、防止出错

  • 做出HTML实现不了的特效

    ​ CSS2很多功能HTML里面其实也有，但在CSS3以及之后HTML里的样式就不再更新了

3. 认识<div> 和<span> 标签(双)

​ <div> 和<span> 标签是属于HTML的标签，一般这两个标签都是配合样式表使用的，这两个标签可以理解为容器标签，早期时一般用Table来当容器

  • <div> 标签(双)：

    ​ 用来为HTML文档内大块的内容提供结构和背景的元素，它为块元素，<div> 的起始标签和结束标签之间的所有内容都在该块中，一般在布局时使用

  • <span> 标签(双)：

    ​ 用来组成HTML文档的行元素，一般放内容时使用，div 也可以作为放内容

​ <div> 和<span> 在页面上的显示

​ 代码

<body>
	<div>内容1</div> <div>内容2</div>
	<span>内容1</span> <span>内容1</span>
</body>

​ 结果(虚线是工具编译生成的，在浏览器显示时，不会进行显示)

  • 选择器(符)：用于选择文档中要应用样式的那些元素，该元素

5. CSS样式表中的注释

​ 使用/* 注释 */包裹进行注释

6. CSS样式的几种表达方式

(1). 内嵌样式(直接在页面的标签里加)

​ 将CSS样式与某HTML标签混合使用，这种方式可以简单的对某个元素单独定义样式

​ 范例：内嵌样式的使用

<ul>
	<li style="font-family:'微软雅黑';font-size:14px;color:#F00">内嵌样式</li>
</ul>

​ 注意：这种方式实际上与在HTML代码中直接定义属性类似，违反了之前避免使用不必要的HTML元素这个规则，一般使用这种方式写只是针对页面已经写好但部分需要修改时使用

(2). 内部样式表(在head部分加入)

​ 在HTML页面head标签内进行定义，只能在定义了的页面进行调用，其他页面调用不了

​ 范例：内部样式表的使用

<head>
	<style> /*有些会写<style type="text/css">这是CSS4.0要求的，现在5.0可以不写*/
		.font_size{
			font-size:30px;
		}
	</style>
</head>
<body>
	<div class="font_size">字体变大</div>
</body>

​ 只适合单页或者是要修改时使用，在修改时，直接修改css文件可能会造成重名，而使用这个方式可以很好的避免

​ 注意：之所以使用会写type="text/css"作为<style> ，是因为HTML的设计者当时认为还有可能会有其他样式，但现在并没有，所以直接写<style>不带属性也是可以的

(3). 链入外部样式表(在head部分进行引用描述)

​ 把所有的东西封装在一个样式表文件里，从外部引用这个样式表文件，当网页加载时会把所有的外部样式先加载到网页中再进行显示，网速再慢也不影响，推荐使用这个形式

​ 范例：链入外部样式表

<head>
	<link href="my.css" rel="stylesheet" type="text/css" media="all" />
</head>

​ rel属性和type属性是两个声明，href中的路径必须要对

​ 注意：写在样式表中的CSS的样式定义与上面的写法是一样的

​ 在浏览器中有的可以设置文字的大小，所以一般在定义文字时用CSS限制文字大小，不然可能造成页面被浏览器弄失真

(4). 导入外部样式表(在head部分进行引用)

​ 导入样式类似于链入外部样式，都是利用外部CSS样式文件来改变页面的外部表现的，但与外部样式不同的是，需要在<style>和</style> 标签中间利用”@import”关键字导入外部CSS文件，文件加载到网页时，先把HTML结构显示出来，再把外部的CSS加载到网页当中，当网速较慢是会先显示未经修饰的HTML页面造成阅读者不好的感觉

​ 现在使用的非常少，这是早期样式表导入的语法

​ 范例：导入外部样式表

<head>
	<style type="text/css">@import url(my.css);</style>
</head>
(5). 样式表的优先级

​ 内嵌样式 > 内部样式表 > 导入外部样式表

7. CSS基本选择器(符)

​ 注意：所写的一个样式可以被多个内容所引用且本身样式表是有继承性的，有优先级别

(1). 通配符选择器(*)

​ 在编码时，用”*”表示通配选择器，其作用是定义页面所有元素的样式，不管是直接在body标签中敲还是放什么标签里

​ 范例：定义页面中所有的元素样式

*{
	margin:0px;
	padding:0px;
}
(2). 类选择器(.)

​ 类选择器通过直接引用元素中类属性的值而产生效果，使用. 来标识一个类选择器，类名可以随意命名但最好根据元素用途来命名

​ 类选择器的语法

//CSS中
.选择器名称(A){
	//样式	
}

//HTML中
<要引用样式的标签名 class="选择器名称(A)">
(3). ID选择器(#)

​ 使用# 来标识一个ID选择器，在HTML文档中，每一个标签都可以使用id=”选择器名称”的形式进行一个名称指派，这和类选择器相似，但需要注意的是，HTML文件中id是具有唯一性是不可以重复的

​ 一般在js脚本中都是认id名的

(4). 标签选择器

​ 把HTML标签当作选择器的名称，所有被该标签包含的HTML标签都会改变样式

(5). 群组选择器

​ 名称之间用, 隔开

​ 范例：让三个div宽高一样，颜色不同

<head>
	<style>
        .a1,.a2,.a3{
			width:300px;
            height:300px;
        }
        
        a1{
            background-color:red;
        }
        
        a2{
            background-color:green;
        }
        
        a3{
            background-color:yellow;
        }
	</style>
</head>

<body>
	<div class="a1">测试1</div>
	<div class="a2">测试2</div>
	<div class="a3">测试3</div>
</body>

8. 伪类(里面的状态伪类)

​ 同一个标签，根据其不同的状态，有不同的样式。这就叫做“伪类”，伪类用冒号来表示。

​ 伪类选择器分为两种。

​ 1. 静态伪类：只能用于**超链接<a> **的样式。如下：

  • :link 超链接点击之前

  • :visited 链接被访问过之后

    注意：以上两种样式，只能用于超链接，且要写的话应该一起写上。

​ 2. 动态伪类：针对所有标签都适用的样式。如下：

  • :hover “悬停”：鼠标放到标签上的时候
  • :active “激活”： 鼠标点击标签，但是不松手时。
  • :focus 是某个标签获得焦点时的样式（比如某个输入框获得焦点）

​ 超链接的四种状态伪类

​ a标签有4种伪类（即对应四种状态），要求背诵。如下：

  • :link “链接”：超链接点击之前

  • :visited “访问过的”：链接被访问过之后

  • :hover “悬停”：鼠标放到标签上的时候

  • :active “激活”： 鼠标点击标签，但是不松手时。

​ 记住，在css中，这四种状态必须按照固定的顺序写：

a:link 、a:visited 、a:hover 、a:active

​ 如果不按照顺序，那么将失效

9. CSS颜色写法

​ 在CSS中，颜色的表达有一下三种方式

  1. 十六进制：如“#ff000”
  2. RGB：如“rgb(255,0,0)”
  3. 颜色名称：如“red”

​ 注意：一般使用十六进制表示颜色这种方式居多

10. 背景图像

​ CSS关于背景图像的属性

  • background-color: blue 为元素设置背景颜色为蓝色
  • background-image:url("图像路径") 设置背景图像
  • background-repeat:no-repeat 设置背景图像为不重复，默认值为重复
  • background-attachment:fixed 将图片滚动锁定为不跟随滚动
  • background-position:center 将图像位置设置为居中

11. 文本修饰

​ CSS关于文本修饰的属性

  • color:blue 设置文本颜色为蓝色
  • text-align:center 设置文本位置为居中
  • text-decoration:line-through 设置文本效果为加删除线，除了删除线外还有上划线、下划线和无文字效果(none，这个可以和超链接混合使用)
  • text-indent:50px 设置文本缩进为50px，通常是将段落的首行缩进，允许缩进使用负值

12. 文字修饰

​ CSS关于文字修饰的属性

  • font-family:字体种类 它可以设置多种字体，但在浏览器显示时需要操作系统中本身有该字体，不然会从浏览器下载该字体，多个字体按顺序来优先显示
  • font-style:italic 将文字样式设置为斜体，默认值为normal即正常显示
  • font-size:40px 将文字设置为40像素大小，还要一个单位em 1em=16px
  • font-weight:bold 将文字设置为加粗(bold=700)，默认值为normal即正常显示，还可以使用数字表示100~900之间

14. CSS表格相关样式的设置

​ 设置表格边框的属性(有边框的容器都适用)

  • border:2px solid blue; 放在table标签中为设置边框的外边框为2像素，实线，颜色为蓝色，放在td标签中，即为设置内边框样式
