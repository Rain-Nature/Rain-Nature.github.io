事件

1. 事件

​ 事件处理是整个JavaScript的核心操作

​ 在页面上任何的操作都可能被称为事件源，而对每一个事件可以自定义处理操作，例如页面加载，鼠标移动等都被称为事件源

​ 在JavaScript中，事件都是以onXXX的形式命名的

​ 最基础的静态事件即在HTML标签中声明需要该事件并定义事件名要加小括号和分号，如XXXX();，而后在JavaScript中定义与事件名相同的函数

​ Javascript是基于对象的编程语言，所有的对象都在元素上，即每一个元素都属于一个JavaScript的对象

2. 页面加载事件与页面卸载事件

​ 注意：这两个事件只能够在<body> 元素中进行处理

(1). 页面加载事件：onload

​ 页面加载完成后执行的动作

​ 页面加载时，大致可以分为以下几个步骤：

​ 1. 开始解析HTML文档结构

​ 2. 加载外部样式表及JavaScript脚本

​ 3. 解析执行JavaScript脚本

​ 4. DOM树渲染完成

​ 5. 加载未完成的外部资源（如 图片）

​ 6. 页面加载成功

​ 范例：onload 的基础使用

<html>
    <head>
    	<meta charset="UTF-8">
    	<title>测试</title>
        <script type="text/javascript">
            function loadHandle(){
                alert("欢迎来到测试页面");
            }
        </script>
    </head>
    <body onload="loadHandle();">
    </body>
</html>

​ 注意：在使用HTML标签中要加多个事件，则在;继续加就可以了

(2). 页面卸载事件：onunload

​ onunload 事件在用户退出页面时发生

​ 范例：使用onunload事件

<html>
    <head>
    	<meta charset="UTF-8">
    	<title>测试</title>
        <script type="text/javascript">
            function unloadHandle(){
                alert("再见");
            }
        </script>
    </head>
    <body onunload="unloadHandle();">
    </body>
</html>

​ 注意：这个代码为实现，应该是浏览器的问题

3. 鼠标事件

事件类型 说明
click 单击鼠标左键时发生，如果右键也按下则不会发生。当用户的焦点在按钮上，并按了回车键时，同样会触发事件
dblclick 双击鼠标左键时发生，如果右键也按下则不会发生
mousedown 单击任意一个鼠标按钮时发生
mouseout 鼠标指针位于元素上，且将要移出元素的边界时发生
mouseover 鼠标指针移入元素时触发
mouseup 松开任意一个鼠标按钮时发生
mousemove 鼠标在某个元素上移动时发生

​ 范例：使用单击点击事件

<html>
    <head>
    	<meta charset="UTF-8">
    	<title>测试</title>
        <script type="text/javascript">
            function test(){
                alert("你好");
            }
        </script>
    </head>
    <body>
        <button onclick="test()">点击</button>
    </body>
</html>

​ 提升：鼠标经过表格变色

<html>
    <head>
    	<meta charset="UTF-8">
    	<title>测试</title>
        <style>
            .table1{
                width: 300px;
                height: 20px;
                border: 1px solid black;
                text-align: center;
                background-color: #f2f2f2;
                
            }

            td{
                border: 1px solid black;
            }

        </style>
        <script type="text/javascript">
            function change(obj,color){
                obj.bgColor = color;
            }
        </script>
    </head>
    <body>
        <table class="table1" cellpadding="10px" cellspacing="0px">
            <tr onmousemove="change(this,'white')">
                <td>姓名</td>
                <td>年龄</td>
                <td>性别</td>
            </tr>
            <tr onmousemove="change(this,'white')">
                <td>张三</td>
                <td>12</td>
                <td>男</td>
            </tr>
            <tr onmousemove="change(this,'white')">
                <td>李四</td>
                <td>15</td>
                <td>男</td>
            </tr>
        </table>
    </body>
</html>

​ 注意：bgcolor是HTML，bgColor是JavaScript

4. 动态设置事件

​ 在JavaScript中，事件的绑定除了在元素上使用onXXX 静态的定义之外，还可以动态的设置

(1).动态设置事件的全过程

​ 范例：使用onload在页面加载时对事件进行处理

<html>
    <head>
    	<meta charset="UTF-8">
    	<title>测试</title>
        <script type="text/javascript">
            function handle(){
                alert("事件触发！");
            }
            //window为JS的内置对象
            window.onload = function(){ //匿名函数
                handle();
            }
        </script>
    </head>
    <body></body>
</html>

​ 注意：所有的动态设置的事件都可以在onload里面进行动态配置，但此时需要取得明确的元素对象，可以使用document.getElementById(元素ID) 的操作方式取得元素对象，此时该方式应该写在onload里面

​ 注意：id是整个JavaScript的操作核心所在，必须存在

​ 然后通过使用元素ID赋予的变量.addEventListener(事件类型，处理函数名称，触发时机) 对该元素进行动态配置事件，事件类型要加双引号，处理函数名称后面不需要括号，触发时机一般设置为false表示在事件的触发过程进行处理，阻止事件冒泡，其也要写在onload里面

​ 动态设置操作事件的好处在于HTML代码不会和JavaScript代码混合在一起

(2).范例：图片的轮转
<html>
    <head>
    	<meta charset="UTF-8">
    	<title>测试</title>
        <style>
            img{
                width: 598px;
                padding: 0px;
                border: 1px solid black;
            }

            div{
                margin-top: 10px;
            }

            button{
                margin-left: 20px;
            }
        </style>
        <script type="text/javascript">
            var imges = new Array("img/01.jpg","img/02.jpg","img/03.jpg","img/04.jpg","img/05.jpg");
            var index = 0;
            
            window.onload = function(){ //匿名函数
                var pic = document.getElementById("pic");
                var next = document.getElementById("next");
                var up = document.getElementById("up");
                next.addEventListener("click",function(){
                    if(index>=4){
                        index = -1;
                    } 
                    pic.src = imges[++index];
               },false)

               up.addEventListener("click",function(){
                    if(index<=0){
                        index = imges.length;
                    }
                    pic.src = imges[--index];
               },false)
            }
        </script>
    </head>
    <body>
            <img id="pic" src="img/01.jpg">

        <div>
            <button id="up">上一张</button>
            <button id="next">下一张</button>
        </div>
    </body>
</html>

5. innerHTML

​ JavaScript可使用document.getElementById(id).innerHTML = “新内容”来修改id元素内的HTML内容

​ 范例：使用innerHTML

<html>
    <head>
    	<meta charset="UTF-8">
    	<title>测试</title>
        <script type="text/javascript">
           function clickb(){
               document.getElementById("a").innerHTML = "修改后的内容";
           }
        </script>
    </head>
    <body>
        <h1 id="a">初始内容</h1>
        <button onclick="clickb();">点击修改内容</button>
    </body>
</html>

​ innerHTML和document.write的区别

1. document.write：是直接写入到页面的内容流，如果在写之前没有调用document.open, 浏览器会自动调用open。每次写完关闭之后重新调用该函数，会导致网页内容全部被重写
 2. innerHTML：是DOM页面元素的一个属性，代表该元素的html内容。（获取div里面的内容 ，修改内容）你可以精确到某一个具体的元素来进行更改，其不会导致页面全部重绘

6. 时钟

​ 在JavaScript中，Date对象用来表示日期和时间

​ 范例：获取时间

<script type="text/javascript">
           var date = new Date();
           document.write(date);
        </script>

​ 在JavaScript中，提供setTimeout(函数，时间-毫秒)函数，该函数是指在指定时间后执行一次指定函数，常用作定时器

​ 范例：实现页面时钟

<html>
    <head>
    	<meta charset="UTF-8">
    	<title>测试</title>
        
    </head>
    <body>
        <h1 id="a"></h1>
    </body>
    <script type="text/javascript">
        function test(){
            var date = new Date();
            document.getElementById("a").innerHTML = date;
            setTimeout(test,1000);//过一秒执行该函数
        }
        test();
    </script>
</html>

​ 注意：此时JavaScript代码放在body后面，如果放前面就无法显示

​ JavaScript放在head和放在body后的区别

1. 放在head中：需调用才执行的脚本或事件触发执行的脚本放在HTML的head部分中。当你把脚本放在head部分中时，可以保证脚本在任何调用之前被加载
 2. 放在body后：当页面被加载时立即执行的脚本放在HTML的body部分。放在body部分的脚本通常被用来生成页面的内容。
