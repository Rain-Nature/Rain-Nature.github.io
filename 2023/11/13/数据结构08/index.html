查找和排序

1. 查找和排序的关系

​ 排序是查找的前提，排序是重点

2. 排序的稳定性

​ 当两个元素值是一样的设定为A和B，A在B的前面，在排完序之后A依然在B的前面则称排序是稳定的，否则排序是不稳定的

3. 排序

  • 冒泡

  • 插入

    从两个元素开始，按照规定插入，保证这两个有序，然后插入第三个元素并保证这三个元素有序，依次类推

  • 选择

    以升序为例，在所有的元素中筛选出最小的，然后与序列中第一个元素交换位置，再从剩余元素中找出最小的与序列的第二个元素交换位置，依次类推

  • 快速排序

  • 归并排序

    两个两个有序，四个四个有序，最后整体有序

4. 选择排序的算法实现-直接选择排序

#include <stdio.h>
#include <stdlib.h>
//直接选择排序
void sort(int * a ,int len){
    int i,j,min,t;

    for(i=0 ; i<len-1 ; ++i){
        //选出最小元素
        for(min=i , j=i+1 ; j<len ; ++j){
            if(a[min] > a[j]){
                min = j;
            }
        }
        //将最小的值与前排元素交换位置
        if(min!=i){
            t = a[i];
            a[i] = a[min];
            a[min] = t;
        }
    }
}

int main()
{
    int a[6] = {4,0,3,2,5,1};
    sort(a,6);
    for(int i=0; i<6 ; i++){
        printf("%d ",a[i]);
    }
    return 0;
}

5. 快速排序

​ 先找到某一个元素的排好序的确切位置，一般假定是第一个元素，把数据变为两半，然后在这两半数据中再重复之前动作，直到完全排好序

​ 快速排序细节上也分为很多种，但整体思路是一样的

​ 当前讲解一种的伪算法

​ 假设升序排序，首先定义两个指针(L、H)分别指向首元素和末尾元素，将首元素的值保存在临时变量(val)中。

​ 先移动h，将h指的值与val里的值作比较，如果h所指的值比val的小，则将h所指的值赋给L所指的值，若比val大则继续移动h；值发生交换则开始移动L，L所指的值与val作比较，如果比val小则继续移动L，如果比val大，则将L所指的指赋给H所指的值，值发生交换则开始移动H；以此类推，最后当L与H相交时，将val赋值给两个指针所指向的节点，以该节点作为分界在分别对截成的两个数组的前后进行快速排序

​ 范例：快速排序算法

#include <stdio.h>
#include <stdlib.h>
//递归调用快速排序
void Quicksort(int * a ,int low , int high);
int findnum(int *a,int low,int high);
void Quicksort(int * a ,int low , int high){
    if(high>low){
        int pos = findnum(a,low,high);
        Quicksort(a,low,pos-1);
        Quicksort(a,pos+1,high);
    }

}
//找出第一个元素完全排好序后应该放在的位置，在该位置放置该元素，并将该位置返回
int findnum(int *a,int low,int high){
    int tmp = a[low];
    while(high>low){
        while(high>low&&tmp<=a[high]){
            high--;
        }
        a[low] = a[high];
        while(high>low&&tmp>=a[low]){
            low++;
        }
        a[high] = a[low];
    }
    a[low] = tmp;
    return low;

}

int main()
{
    int a[6] = {3,0,3,-2,5,1};
    Quicksort(a,0,5);
    for(int i=0; i<6 ; i++){
        printf("%d ",a[i]);
    }
    return 0;
}

6. 全课总结

​ 再次讨论什么是数据结构？

​ 答：数据结构是研究数据的存储和数据操作的一门学问

​ 数据的存储分为哪两个部分？

​ 答：个体的存储

​ 个体关系的存储

​ 从某个角度而言，数据的存储最核心的就是个体关系的存储，个体的存储可以忽略不计

​ 再次讨论什么是泛型

​ 答：同一种逻辑结构，无论该逻辑结构物理存储是什么样子的，我们可以对它执行相同的操作
